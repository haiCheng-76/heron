### 项目说明
* 基于SpringBoot的网站内容管理系统
* 可视化的管理定时任务
* model、mapper、mapper.xml代码生成器
* 系统运行监控

### 功能概述
* 用户管理
* 角色管理
* 定时任务管理
* 基于MinIO的文件管理
* 基于Quartz的定时任务管理
* 登录日志管理
* 邮箱重置密码
